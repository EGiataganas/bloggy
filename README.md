# Bloggy

| Dependency | Version |
|:-----------|:--------|
| Ruby       | 2.7.1 |
| Rails      | 6.0.3.3 |

## Getting Started

1.  Clone the project

  `git clone git@github.com:EGiataganas/bloggy.git`


2. Install the project's dependencies using bundler

  `bundle install`

## Server

  `rails s`
